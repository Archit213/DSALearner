C - Linked List Data Structure

C - Linked List Implementation

In C, linked lists are constructed using structures. Each node contains data and a pointer to the next node: 

// Define the structure for a linked list node
struct Node {
    int data;
    struct Node* next;
};

// Initialize a new node
struct Node* head = NULL;
head = (struct Node*) malloc(sizeof(struct Node));
head->data = 1;
head->next = NULL;

Addition of Elements

To add elements, you link new nodes to the existing list:

struct Node* newNode = (struct Node*) malloc(sizeof(struct Node));
newNode->data = 2;
newNode->next = NULL;
head->next = newNode;

Deletion of Elements

You can delete a node by adjusting the links and freeing memory:

struct Node* temp = head;
head = head->next;
free(temp);

Python - Linked List Implementation

In Python, linked lists can be implemented using classes. Each node contains data and a reference to the next node:

# Define the class for a linked list node
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

head = Node(1)
