C - Stack Data Structure

C - Stack Operations

In C, stacks can be implemented using arrays. Below is an example of basic stack operations: 

// C program to implement stack operations
#define MAX 100 // Maximum size of stack

int stack[MAX];
int top = -1; // Initialize top of stack

void push(int item) {
  if (top == MAX - 1) {
    // Stack overflow
    return;
  }
  stack[++top] = item;
}

int pop() {
  if (top == -1) {
    // Stack underflow
    return -1;
  }
  return stack[top--];
}

C++ - Stack Operations

In C++, stacks can be implemented using the Standard Template Library (STL). Here's an example: 

// C++ program to demonstrate stack operations
#include <stack>
using namespace std;

int main() {
  stack<int> s;

  s.push(10);
  s.push(20);
  s.push(30);

  // Popping elements
  while (!s.empty()) {
    cout << s.top() << endl;
    s.pop();
  }
  return 0;
}

Python - Stack Operations

In Python, stacks can be implemented using lists. Below is an example: 

# Python program to demonstrate stack operations
stack = []

def push(item):
    stack.append(item)

def pop():
    if stack:
        return stack.pop()
    return None

# Example usage
push(10)
push(20)
push(30)

while stack:
    print(pop())
