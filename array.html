C - Array Data Structure

C - Array Initialization

In C, arrays are used to store multiple values of the same data type in contiguous memory locations. Below is an example of initializing an array in C: 

// Declaring and initializing an array of 5 integers
int arr[5]; // Declares an array of 5 integers
arr[0] = 1; // Assigns value 1 to the first element
arr[1] = 2; // Assigns value 2 to the second element

Addition of Elements

You can loop through the array to assign values or modify them:

for (int i = 0; i < 5; i++) {
    arr[i] = i + 1; // Assigns values from 1 to 5
}

Deletion of Elements

In C, you can "delete" an element by setting its value to 0 or another placeholder:

arr[2] = 0; // Deletes the third element (sets it to 0)

C++ - Array Initialization

In C++, arrays work similarly to C. Here's how you can declare and initialize an array: 

// Declaring and initializing an array of 5 integers in C++
int arr[5];
arr[0] = 1;
arr[1] = 2;

Addition of Elements

In C++, you can use loops similarly to C:

for (int i = 0; i < 5; i++) {
    arr[i] = i + 1;
}

Deletion of Elements

You can "delete" or reset elements similarly:

arr[2] = 0;

Python - Array Initialization

In Python, arrays (or lists) are dynamically typed and easy to manipulate: 

# Declaring and initializing an array (list) in Python
arr = [1, 2, 3, 4, 5]

Addition of Elements

You can append new elements to a Python list using append(): 

arr.append(6);

Deletion of Elements

To delete an element from a Python list, you can use the remove() method or the del statement: 

# Deleting an element with value 3
arr.remove(3);

# Deleting an element at index 2
del arr[2];
